适配 id 选择器 color_blue

head 中的 tag 元素 meta 元数据 使用 charset 属性指明当前 html 的编码方式,使用 name content 属性添加额外数据,name 的值可以为 author,description 等关键key值,content 则为该key值对应的内容.

description 的 key 值作用通常是描述该网站的实际内容,便于搜素引擎抓取,显示在搜素结果概览页面中便于搜素用户了解当前网页.

FaceBook 定义了元数据协议 Open Graph Data Twitter 同样定义有自己的元数据展示协议.

在 Head 中可以使用 link 标签加载 css rel (Relation) 指定关系为 stylesheet type 指定为 text/css href 表示引用的 css 样式表的文件链接

script 标签 不一定需要放置在 head 中,其会阻止元素的正常加载.通常可以将 script body 结束标签之前,这样会先加载body中的元素样式显示,再加载script 标签中的内容. script 标签可以放置在 body 结束标签之前的任何位置.

html 标签以及其他标签可以设置 lang 属性标记,属性值可以参考 ISO 639-1 语言tag描述文档
